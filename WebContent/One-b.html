Html Overview

HTML (Hypertext Markup Language) uses a markup system composed of elements which represent speciﬁc content. Markup means that with HTML you declare what is presented to a viewer, not how it is presented. . Visual representations are deﬁned by Cascading Style Sheets (CSS) and realized by browsers.
